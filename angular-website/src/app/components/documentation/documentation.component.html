Introduction

DevStart is a platform that enables developers to initiate a full-stack web project (backend + frontend) in minutes, directly from an existing SQL or NoSQL database. The goal is to streamline and accelerate the setup phase of new software projects by automating repetitive development tasks.

Key Objectives

  • Accelerate project initialization by automating code generation
  • Simplify the backend & frontend setup with no manual configuration
  • Focus developers' efforts on value-added features
  • Reduce human error and maintain project consistency
  • Support both beginner and experienced developers

Key Concepts & Definitions

  • CRUD: Create, Read, Update, Delete
  • API: Interface allowing components to communicate
  • Frontend: The user interface
  • Backend: Server-side logic and database access
  • MongoDB: NoSQL document-based database
  • MySQL: Relational database management system
  • Docker: Containerization tool
  • Scripts: Shell/batch commands to automate code generation

Getting Started Guide

  1. Open the DevStart Platform
  2. Select frontend , backend and database
  3. Enter connection parameters (host, port, username, password, database name)
  4. Click ‚ÄúGenerate Project‚Äù
  5. Download the generated .zip archive
  6. Run your project:

Backend:

cd backend
npm install
node index.js

Frontend:

cd frontend
npm install
ng serve
